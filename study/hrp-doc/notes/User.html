# 获取个人用户信息

说明：获取个人用户信息

获取方式： HTTP GET

发送参数:

{
	"uid": 10001
}

返回数据格式: JSON

接口地址(模拟)：http://localhost:3000/personInfo

调用例子: http://localhost:3000/personInfo?uid=10001

返回示例:

{
    "code": 200,
    "success": true,
    "msg": null,
    "data": {
        	"uid": 10001,
            "username": "lv",
            "age": 18,
        	"sex": "男",
            "phone": "15816990000",
            "email": "lyuxxx@foxmail.com",
            "powerId": 1
        }
}

# 修改个人信息

说明：修改个人信息

获取方式： HTTP POST

发送参数:

{
	"uid": 10001, 
	"username": "lv", 
	"sex": "男", 
	"age": 22 , 
	"phone": "110", 
	"email": "110@qq.com", 
	"powerId": 1, 
	"pwd": "123" //判断是否为空去决定是否改密码
}

返回数据格式: JSON

接口地址(模拟)：http://localhost:3000/personInfo

调用例子: http://localhost:3000/personInfo

返回示例:

{
    "success": true,
    "code": 200,
    "msg": "保存成功"
}

{
    "success": false,
    "code": 200,
    "msg": "保存失败"
}

# 搜索用户信息

说明：根据用户名或职工号查询数据

获取方式： HTTP POST

发送参数:

{
	"user": 10001 //或用户名
}

返回数据格式: JSON

接口地址(模拟)：http://localhost:3000/searchInfo

调用例子: http://localhost:3000/searchInfo

返回示例:

{
    "success": false,
    "code": 200,
    "msg": "查询无结果"
}

{
    "success": true,
    "code": 200,
    "msg": "null",
